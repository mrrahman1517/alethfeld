\section{Supporting Lemma}

The proof relies on the following well-known result:

\begin{lemma}\label{lem:even-square}
For any integer $n$: if $n^2$ is even, then $n$ is even.
\end{lemma}

\begin{proof}
We prove the contrapositive. Suppose $n$ is odd. Then $n = 2m + 1$ for 
some integer $m$. Thus:
\[
n^2 = (2m+1)^2 = 4m^2 + 4m + 1 = 2(2m^2 + 2m) + 1
\]
which is odd. Therefore, if $n$ is odd, then $n^2$ is odd.
By contraposition, if $n^2$ is even, then $n$ is even.
\end{proof}
